# 🎤 GroqTalk

An AI-powered voice assistant built using the Groq API with speech recognition and text-to-speech capabilities.

---

## ✨ Features

- 🎙️ Speech Recognition
- 🤖 AI Chat using Groq API
- 🔊 Text-to-Speech
- 💬 Interactive Chat Interface
- ⚡ Fast AI Responses
- 🎨 Modern User Interface

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/adarshts101/GroqTalk.git
```

2. Navigate to the project folder

```bash
cd GroqTalk
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Create a `.env` file

```env
GROQ_API_KEY=your_api_key_here
```

5. Run the application

```bash
python main.py
```
